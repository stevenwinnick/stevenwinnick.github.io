Walking Robot

As part of Columbia's Robotics Studio this semester, I've built a walking robot. Nicknamed the "Trice-Robot," this triceratops-fossil-shaped robot started as a hand-sketched design, which Rex Li and I then modeled in its entirety in SolidWorks, down to every last screw and insert. After 3D-printing and assembling the robot, we cut up and soldered some cables to wire the electronics and installed an onboard computer. We configured the computer to send us an email with it's IP Address upon startup, allowing us to control our robot remotely with an SSH connection. I did some math to figure out how to rotate the servo motors to give us a smooth walking path, then wrote some error-handling code to control all 8 servos simultaneously and let our robot walk. In the coming weeks, I'll be using PyBullet's Machine Learning software to optimize our robot's walking speed, as well as installing a camera to implement object detection using Computer Vision.

Neural Network From Scratch

After my internship ended this summer, I wanted to get a deeper understanding of deep learning, specifically neural networks and how they work, so I built one from scratch using Python. My neural net uses a sigmoid activation function for an added computational challenge and manually calculates and applies partial derivatives for backpropagation. It can also scale to contain any number of hidden layers, each of any size, that it is initialized with, and accepts both batches and individual inputs. You can find the code for this project here.
